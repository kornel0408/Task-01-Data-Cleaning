# Task 1: Data Cleaning and Preprocessing

## Internship Details

**Program:** Elevate Labs Data Analyst Internship

**Task:** Data Cleaning and Preprocessing

## Objective

The objective of this task is to clean and prepare a raw dataset by identifying and handling common data quality issues such as missing values, duplicate records, inconsistent formatting, and incorrect data types.

## Dataset

Netflix Movies and TV Shows Dataset

## Tools Used

* Python
* Pandas
* Jupyter Notebook

## Tasks Performed

* Imported and loaded the dataset
* Explored dataset structure and data types
* Standardized column names
* Identified and handled missing values
* Checked and removed duplicate records
* Cleaned text data by removing extra spaces
* Converted date columns to datetime format
* Verified data types
* Performed final data quality checks
* Exported the cleaned dataset

## Files Included

* Raw Dataset
* Cleaned Dataset
* Jupyter Notebook
* Task Summary

## Outcome

The dataset was successfully cleaned and transformed into a structured format suitable for further analysis and visualization.
